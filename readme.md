# Compiler Project

## Build Instructions

1. **Clone the repository:**
    ```sh
    git clone <repository-url>
    cd Compiler
    ```

2. **Build the project:**
    ```sh
    mkdir build
    cd build
    cmake ..
    make
    ```

3. **Run the compiler:**
    ```sh
    ./compiler
    ```

## Requirements

- C++ compiler (e.g., g++)
- CMake
